* Problem Statement
Spring Flowers School wants to manage student information online.
This application allows students to view their Time Table easily on desktop and mobile devices.

* User Stories Implemented
1. As a Student, I can see the TimeTable button in the menu
2. As a Student, I can view my Time Table after clicking the button
3. As a Student, I can open and use the application on mobile devices

* Technologies Used
1. HTML5 – Page structure
2. CSS3 – Styling and responsiveness
3. Flexbox – Layout arrangement
4. Bootstrap 5 – UI components (buttons, table, navbar)
5. HTTP Server – Application deployment

* Features
1. Responsive navigation menu
2. Time Table button visible in menu
3. Clean and simple Time Table layout
4. Mobile-first design approach
